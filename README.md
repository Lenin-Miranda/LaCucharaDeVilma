Nicaraguan homemade restaurant in Las Vegas, built with Next.js, React, and Tailwind CSS.

## Description

La Cuchara de Vilma is a website for a family restaurant celebrating Nicaraguan cuisine in Las Vegas. Here you can learn our story, see photos, read customer reviews, and find out how to visit or contact us.

## Main Technologies

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript
- AOS (Animate On Scroll)
- React Icons

## Project Structure

- `src/app/components/` ‚Äî Main components (Hero, About, Reviews, Visit, Footer, NavBar, etc.)
- `public/images/` ‚Äî Restaurant, dishes, and customer images
- `src/app/globals.css` ‚Äî Global styles
- `tailwind.config.js` ‚Äî Tailwind configuration

## Installation & Local Usage

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd la-cuchara-de-vilma
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
